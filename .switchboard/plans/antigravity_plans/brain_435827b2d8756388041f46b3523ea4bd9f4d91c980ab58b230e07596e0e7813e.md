# 📄 แผนการเปลี่ยน Library PDF เป็น pdf-lib

เปลี่ยนระบบสร้างเอกสาร ปพ.1 จาก `pdfmake` (High-level) เป็น `pdf-lib` (Low-level) เพื่อความยืดหยุ่นในการจัดวางองค์ประกอบและรองรับฟอนต์ภาษาไทยได้ดียิ่งขึ้น

## User Review Required

> [!IMPORTANT]
> การเปลี่ยนเป็น `pdf-lib` จะต้องมีการคำนวณตำแหน่ง (X, Y) ของตัวหนังสือและเส้นตารางด้วยตนเองทั้งหมด ซึ่งอาจทำให้ใช้เวลาในการปรับแต่ง Layout ให้เหมือนเดิมระยะหนึ่งครับ
